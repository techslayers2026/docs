# Download report

Downloads the report output as a ZIP file.

## Example

```bash
curl -L "https://darkrecon.1337807.xyz/download/YOUR_REPORT_NAME" \
  -H "X-API-Key: $DARKRECON_API_KEY" \
  -o darkrecon-report.zip
```
